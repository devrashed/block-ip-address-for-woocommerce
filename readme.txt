== Frequently Asked Questions ==

= How can I block an IP address in WooCommerce? =
Simply add the IP address from the plugin settings, choose a redirect page, and save changes.

= Can I set a duration for blocked IPs? =  
Yes. You can define a **start and end date** for each blocked IP. The restriction will only be active within this time period.

= Can I redirect blocked users to a custom page? =  
Yes, you can select any page on your website for redirection.

= Does this block access to the entire WooCommerce site? =  
No, you can choose to block only the shop page, homepage, or specific categories.

= Can I block IP ranges? =  
Currently, only individual IP addresses are supported. IP range support may be added in future updates.

= Does this affect website performance? =  
No, the plugin is lightweight and optimized for speed.

= Is it compatible with caching plugins? =  
Yes, the plugin is compatible with most major caching plugins.

= Can I block IPs temporarily? =
Yes, you can set start and end dates for each IP block.
